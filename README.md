Simple starter pack for typescript based node projects.
Uses typescript, eslint (airbnb rules), husky and prettier for maintaing code quality.

## Prerequisite

- nodejs 16>=

## Getting started

### Install

```
npm install
```

### Run

Build the project

```
npm run build
```

Run the project for production

```
npm run start
```

Run the project for development

```
npm run start:dev
```
